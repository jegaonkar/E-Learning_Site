Break and Continue

In Python, the break and continue statements are used within loops to control the flow of the loop.

1. break Statement:

The break statement is used to exit a loop prematurely. When a break statement is encountered inside a loop, the loop is immediately terminated, and program control is transferred to the code after the loop.

Usage: 

for variable in sequence:
		if condition:
			break  # Exit the loop
		# Code inside the loop
	# Code after the loop

Example: 

numbers = [1, 2, 3, 4, 5]

		for num in numbers:
			if num == 4:
				print("Found 4")
				break
			print(num)

In this example, the loop will terminate as soon as it finds the number 4, and the code after the loop will be executed.

2. continue Statement:

The continue statement is used to skip the current iteration of a loop and move to the next iteration. When a continue statement is encountered inside a loop, the remaining code within the current iteration is skipped, and the loop continues with the next item in the sequence.

Usage: 

for variable in sequence:
		if condition:
			continue  # Skip the current iteration
		# Code inside the loop
	# Code after the loop

Example: 

numbers = [1, 2, 3, 4, 5]

		for num in numbers:
			if num % 2 == 0:
				continue  # Skip even numbers
			print(f"Odd number: {num}")

In this example, the continue statement skips even numbers, and only odd numbers are printed.

Both the break and continue statements are useful for controlling the flow of loops and making your code more efficient and flexible. They are commonly used in various programming scenarios to handle specific conditions or requirements within a loop.
