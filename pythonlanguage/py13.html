Lists

In Python, a list is a versatile and widely used data structure that allows you to store and manage collections of items. Lists are ordered and mutable, meaning you can change their contents after they are created. Here's an overview of how to work with lists in Python:

Creating Lists:

You can create a list by enclosing a sequence of items in square brackets [], separated by commas. Lists can contain elements of different data types, including numbers, strings, or even other lists.

Example: 

fruits = ["apple", "banana", "cherry"]
		numbers = [1, 2, 3, 4, 5]
		mixed_list = [1, "apple", True, [2, 3, 4]]

Accessing Elements:

You can access individual elements in a list using their index. Indexing starts at 0 for the first element, and negative indices count from the end of the list.

Example:

fruits = ["apple", "banana", "cherry"] first_fruit = fruits[0] # "apple" last_fruit = fruits[-1] # "cherry"

Modifying Lists:

Lists are mutable, so you can change their content by assigning new values to elements, appending elements, removing elements, or using various list methods.

Example: 

fruits = ["apple", "banana", "cherry"]
		fruits[1] = "orange"  # Change the second element
		fruits.append("strawberry")  # Add an element to the end
		del fruits[0]  # Remove the first element

List Methods:

Python provides a variety of methods to work with lists, such as append(), insert(), remove(), pop(), sort(), and len(). These methods allow you to perform operations on lists efficiently.

Example: 

numbers = [3, 1, 2, 5, 4]
		numbers.sort()  # Sort the list in ascending order
		numbers.append(6)  # Add 6 to the end
		numbers.remove(3)  # Remove the first occurrence of 3
		length = len(numbers)  # Get the length of the list

Iterating Over Lists:

You can use a for loop to iterate over the elements in a list, performing actions on each element.

Example: 

fruits = ["apple", "banana", "cherry"]
		for fruit in fruits:
			print(fruit)
