String Methods and Formatting

Working with strings in Python often involves using various string methods for manipulation and formatting. Here's an overview of common string methods and string formatting techniques:

String Methods:

  • str.upper() and str.lower():

    These methods convert a string to all uppercase or lowercase letters, respectively.

    text = "Hello, World"
	uppercase = text.upper()  # "HELLO, WORLD"
	lowercase = text.lower()  # "hello, world"
  • str.strip(), str.lstrip(), and str.rstrip():

    These methods remove whitespace characters from the beginning and/or end of a string.

    text = "  Python  "
	stripped = text.strip()  # "Python"
  • str.replace():

    This method replaces a substring with another substring in a string.

    text = "I love Python"
	new_text = text.replace("Python", "programming")  # "I love programming"
  • str.split():

    This method splits a string into a list of substrings based on a specified delimiter (default is space).

    text = "apple, banana, cherry"
	fruits = text.split(", ")  # ['apple', 'banana', 'cherry']
  • str.join():

    This method joins a list of strings into one string, using the original string as a separator.

    fruits = ['apple', 'banana', 'cherry']
	text = ", ".join(fruits)  # "apple, banana, cherry"
  • str.find() and str.index():

    These methods search for a substring in a string and return its index. find() returns -1 if the substring is not found, while index() raises an error.

    text = "Python is fun"
	position1 = text.find("is")  # 7
	position2 = text.index("fun")  # 11
  • str.startswith() and str.endswith():

    These methods check if a string starts or ends with a specified substring, returning a Boolean value.

    text = "Hello, World"
	starts_with_hello = text.startswith("Hello")  # True
	ends_with_world = text.endswith("world")  # False

    • String Formatting:

  • f-strings:

    Introduced in Python 3.6, f-strings provide a convenient way to embed expressions within strings. Place an 'f' or 'F' before the string and use curly braces {} to enclose the expressions.

    name = "Alice"
		age = 30
		message = f"Hello, {name}! You are {age} years old."
  • str.format():

    This method allows you to insert values into a string using placeholders within the string and providing the values as arguments to the format() method.

    name = "Bob"
		age = 25
		message = "Hello, {}! You are {} years old.".format(name, age)
  • % Operator:/b>

    String formatting can also be achieved using the % operator. This method is older and less recommended but still supported.

    name = "Carol"
		age = 35
		message = "Hello, %s! You are %d years old." % (name, age)

    • String methods and formatting techniques are essential for manipulating and presenting text data in Python. These tools allow you to create dynamic, readable, and well-structured strings for various applications, such as user interfaces and data processing.
