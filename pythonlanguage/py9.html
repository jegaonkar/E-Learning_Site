Loops

Loops in Python are used to execute a block of code repeatedly. Python provides two main types of loops: the for loop and the while loop. These loops are fundamental for performing repetitive tasks and iterating over data structures. Here's an overview of how to use loops in Python:

1.1. for Loop:

The for loop is used to iterate over a sequence (such as a list, tuple, string, or range) and execute a block of code for each item in the sequence.

2.Syntax: 

for variable in sequence:
		# Code to be executed for each item in the sequence

Example: 

fruits = ['apple', 'banana', 'cherry']

		for fruit in fruits:
			print(fruit)

2. while Loop:

The while loop is used to repeatedly execute a block of code as long as a specified condition is true.

Syntax: 

while condition:
		# Code to be executed as long as the condition is true

Example: 

count = 1

	while count <= 5:
		print(count)
		count += 1

Loop Control Statements:

Python provides loop control statements to modify the behavior of loops:

  • break: Terminates the loop prematurely and exits it.
  • continue: Skips the current iteration and proceeds to the next iteration of the loop.
  • else with for or while: An optional else block can be added to a loop. It will execute after the loop completes normally (i.e., without encountering a break statement).

    • Example with Loop Control Statements: 

    numbers = [1, 2, 3, 4, 5, 6]

		for num in numbers:
			if num == 4:
				print("Found 4")
				break
			elif num % 2 == 0:
				print(f"Even number: {num}")
			else:
				continue
		else:
			print("Loop completed without encountering a 'break' statement.")

    Nested Loops:

    You can nest loops within other loops to handle more complex tasks, like iterating through multi-dimensional data structures.

    Example of Nested Loop: 

    for i in range(3):
		for j in range(3):
			print(f"({i}, {j})")

    Loops are essential for performing repetitive tasks, processing data, and iterating through collections in Python. They allow you to automate tasks efficiently and make your code more versatile and powerful.
