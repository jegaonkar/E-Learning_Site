Conditional Statements

Conditional statements in Python are used to make decisions in your code based on whether certain conditions are met. Python provides several conditional constructs, including if, elif (short for "else if"), and else, to control the flow of your program. Here's an overview of how to use conditional statements in Python:

1.if Statement:

The if statement is used to execute a block of code if a specified condition is true. 

if condition:
		# Code to execute if the condition is true

age = 18 if age >= 18: print("You are an adult.") 

age = 18

		if age >= 18:
			print("You are an adult.")

2. if and else Statements:

The if statement can be combined with an else statement to execute different code when a condition is true or false. 

if condition:
    # Code to execute if the condition is true
else:
    # Code to execute if the condition is false

Example: 

age = 15

		if age >= 18:
			print("You are an adult.")
		else:
			print("You are not yet an adult.")

3. if, elif, and else Statements:

The elif statement (short for "else if") allows you to check multiple conditions. You can have as many elif statements as needed. 

if condition1:
		# Code to execute if condition1 is true
	elif condition2:
		# Code to execute if condition1 is false and condition2 is true
	else:
		# Code to execute if neither condition1 nor condition2 is true

Example: 

grade = 75

		if grade >= 90:
			print("A")
		elif grade >= 80:
			print("B")
		elif grade >= 70:
			print("C")
		else:
			print("F")

4. Nested if Statements:

You can nest if statements within other if statements to handle more complex conditions.

Example: 

x = 10
		y = 5
		
		if x > 5:
			if y > 2:
				print("Both conditions are true.")
			else:
				print("The second condition is false.")
		else:
			print("The first condition is false.")

5. Logical Operators:

You can use logical operators such as and, or, and not to combine conditions in your if statements.

Example: 

age = 25
		income = 60000
		
		if age >= 18 and income > 50000:
			print("You are eligible for a loan.")

Conditional statements are fundamental for making decisions and controlling the flow of your Python programs. They allow you to execute different code blocks based on various conditions, making your code more flexible and responsive.
