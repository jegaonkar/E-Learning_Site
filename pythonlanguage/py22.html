Custom Exceptions

In Python, you can define custom exceptions by creating new classes that inherit from the built-in Exception class or its subclasses. Custom exceptions allow you to handle specific errors or exceptional conditions in your code more effectively. Here's how to create and use custom exceptions:

1. Define a Custom Exception Class:

Create a new class that inherits from Exception or another built-in exception class. You can add your own attributes and methods to the class to provide additional information or functionality. 

class MyCustomError(Exception):
		def __init__(self, message):
			super().__init__(message)
			self.error_code = 1001  # Custom attribute

2. Raise the Custom Exception:

In your code, when you encounter a situation that warrants the use of your custom exception, raise it using the raise statement. 

def my_function(value):
		if value < 0:
			raise MyCustomError("Value cannot be negative")

3.Handling Custom Exceptions:

You can catch and handle your custom exceptions using the try and except blocks, just like you would with built-in exceptions. 

try:
		my_function(-5)
	except MyCustomError as e:
		print(f"Custom error: {e}")

4. Custom Attributes and Methods:

You can include custom attributes and methods in your custom exception class to provide additional information or behaviors. 

try:
		my_function(-5)
	except MyCustomError as e:
		print(f"Custom error: {e}")
		print(f"Error code: {e.error_code}")

5. Hierarchy of Custom Exceptions:

You can create a hierarchy of custom exceptions by defining multiple exception classes that inherit from one another. This allows you to handle related exceptions differently in your code. 

class MyValidationError(MyCustomError):
		def __init__(self, field):
			super().__init__(f"Validation error in field {field}")

Custom exceptions are useful when you need to create meaningful and specific error messages for your code. They make it easier to identify and handle exceptional conditions and provide valuable information for debugging and troubleshooting.
