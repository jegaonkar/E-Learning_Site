Reading Writing

Reading and writing data is a fundamental aspect of programming, especially when working with files and external data sources in Python. Here's an overview of how to read and write data in Python:

Reading Data:

  • Open a File:

    To read data from a file, you need to open the file using the open() function. You provide the file's path and specify the mode as "r" (read).

    with open('data.txt', 'r') as file:
    data = file.read()
  • Read Methods:

  • read(): Reads the entire content of the file as a single string.
  • readline():Reads one line at a time.
  • readlines():Reads all lines into a list.

    • with open('data.txt', 'r') as file:
    line = file.readline()
    lines = file.readlines()

    1. Positional Arguments:

  • Open a File:

    To write data to a file, open the file in write mode ("w"). If the file doesn't exist, it will be created. If it does exist, its content will be truncated.

    with open('output.txt', 'w') as file:
    file.write("Hello, World!")
  • Append Mode:

    To add data to an existing file without overwriting its content, open the file in append mode ("a").

    with open('output.txt', 'a') as file:
    file.write(" - Additional data.")
  • Writing Lines:

    You can write a list of lines to a file using the writelines() method. Remember to add newline characters if needed.

    lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
	with open('output.txt', 'w') as file:
		file.writelines(lines)

    • File Handling with Context Managers:

    Using the with statement as shown in the examples is a best practice. It ensures that the file is properly closed after use, even if an error occurs.

    Exception Handling:

    When reading or writing files, it's good practice to handle exceptions, such as FileNotFoundError and PermissionError, to deal with potential issues gracefully. 

    try:
		with open('data.txt', 'r') as file:
			data = file.read()
	except FileNotFoundError:
		print("The file does not exist.")
	except PermissionError:
		print("Permission to read the file is denied.")

    Reading and writing data in Python is essential for various tasks, including data processing, log files, configuration files, and more. It's important to follow best practices to handle files and exceptions effectively and ensure data integrity.
