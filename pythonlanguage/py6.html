Data Types

Python supports several data types that allow you to work with different kinds of data. Here are some of the most common data types in Python:

1. Integers (int):

Whole numbers, both positive and negative.
Example: age = 30

2.Floating-Point Numbers (float):

Numbers with decimal points.
Example: height = 1.75

3.Strings (str):

Text data enclosed in single (' ') or double (" ") quotes.
Example: name = "Alice"

4.Booleans (bool):

Represents either True or False, typically used in conditional statements.
Example: is_student = True

5.Lists:

Ordered collections of values that can be of different data types.
Example: fruits = ['apple', 'banana', 'cherry']

6.Tuples:

Ordered collections like lists but are immutable, meaning their contents cannot be changed after creation.
Example: point = (3, 4)

7.Dictionaries (dict):

Sets:
Unordered collections of unique values.
Example: unique_numbers = {1, 2, 3, 4, 5}

8.NoneType (None):

Represents the absence of a value or a null value.
Example: result = None

9.Complex Numbers (complex):

Numbers in the form a + bj, where a and b are real numbers, and j represents the square root of -1.
Example: z = 2 + 3j

10.Bytes and Byte Arrays:

Used to represent binary data or sequences of bytes.
Examples: data = b'hello' (bytes), byte_array = bytearray([65, 66, 67]) (byte array)

These data types allow you to work with a wide range of information and perform various operations in Python. Understanding when and how to use each data type is essential for effective programming and data manipulation in Python.
