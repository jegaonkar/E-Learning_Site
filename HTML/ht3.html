HTML Syntax and Structure


In this lesson, we will delve into the fundamental syntax and structure of HTML. HTML (Hypertext Markup Language) is the backbone of web development, and understanding its syntax is crucial for creating web pages.

1. Understanding HTML Elements:

HTML documents are composed of elements, which are represented by tags. These elements define the structure and content of a web page. Elements can be as simple as a line of text or as complex as a multimedia object.

2. Tags and Attributes:

HTML tags are used to enclose elements. Tags come in pairs: an opening tag and a closing tag, with the element's content placed between them. For example, a basic heading element is defined using the < h1 > and < /h1 > tags. Additionally, HTML elements often come with attributes, which provide extra information about the element. For example, the href attribute is used with anchor tags to specify the URL a link should point to.

3. Basic Document Structure:

HTML documents have a specific structure that should be followed. This structure consists of:

'' declaration: Specifies the HTML version and should be the first line in your Document
'< html >' element: The root element that encloses all other elements on the page.
'< head > 'section: Contains meta-information about the document and may link to external resources like stylesheets or scripts.
'< body >' section: Contains the visible content of the web page, such as text, images, and multimedia.

4. Examples:


Let's look at some practical examples to better understand HTML elements and how they are used. For instance, we can create a heading using the < h1 > tag: 

< h1 >This is a Heading< /h1 >

And paragraphs can be created using the < p > tag: 
< p >This is a paragraph of text.< /p >

Attributes and Values
In addition to tags, HTML elements often come with attributes that allow you to customize their behavior. Attributes are defined within the opening tag and have values. For example, the src attribute in an image tag specifies the image file's source: 
< img src="image.jpg" alt="A beautiful image" >

Nesting:
HTML elements can be nested inside one another. This means you can place one element inside another to create a hierarchy or structure within your web page. For example, you can place a paragraph (< p>) element inside a < div > element. 
< div>
    < p>This is a paragraph inside a div.< /p>
< /div>
Comments:
You can add comments in your HTML code to provide explanations or reminders to yourself or other developers. Comments are not displayed in the web browser. 
< !-- This is an HTML comment. -->
Indentation:
While not required, proper indentation is essential for code readability. It helps you and others understand the structure of your HTML document. You can use spaces or tabs for indentation. 
< !DOCTYPE html>
    < html>
        < head>
            < title>My Web Page< /title>
        < /head>
        < body>
            < h1>Welcome to My Page< /h1>
            < p>This is a sample paragraph.< /p>
        < /body>
    < /html>
