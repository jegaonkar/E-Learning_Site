Text and Formatting


In this lesson, we will explore the fundamentals of adding text to a web page using HTML. Text is a fundamental component of web content, and understanding how to structure and format it is essential for effective web development.

1. Adding Text to Your Page:

  • Creating Text Elements: HTML offers several tags to create text elements. These include:
  • Headings (< h1> to < h6>): Used for defining headings or subheadings. < h1> is the highest level, typically representing the main title, and < h6> is the lowest.

  • Paragraphs (< p>): Used for structuring and formatting paragraphs of text.

  • Inline Text Elements: Tags like < span>, < strong>, < em>, and others allow you to style specific portions of text within paragraphs or headings.

  • Text Formatting: HTML provides basic text formatting options through tags.For example:

  • Font Size (< font size="">): You can change the size of text using this attribute. However, using CSS is the recommended way for text styling.

  • Font Color (< font color="">): This attribute lets you change the color of text. Again, CSS is the preferred method.

  • Text Alignment (< p align="">): You can use this attribute to control the alignment of text within a paragraph. CSS is a better choice for alignment.

  • Semantic Elements: HTML5 introduced semantic elements like < header>, < footer>, < article>, and < section, which provide meaning to the structure of a web page. Using these elements helps with accessibility and SEO. For instance, < header> is used to define the header section of a webpage, and < footer> defines the footer section.

    • 2. Headings and Paragraphs:

    This lesson focuses on two fundamental text elements in HTML: headings and paragraphs.
    Headings: HTML provides six levels of headings (< h1> to < h6). These are used to structure content hierarchically. < h1> is the highest level, typically representing the main title or heading of a webpage, while < h6> is the lowest and usually used for subheadings.

    Paragraphs: The < p> tag is used to create paragraphs. It's essential for structuring text content, making it more readable and organized. You can apply CSS styles to paragraphs to control their appearance, such as line spacing and margins.

    Structuring Content: Effective web content often combines headings and paragraphs to create a well-structured page. Headings provide a clear hierarchy, and paragraphs are used for in-depth content. Proper use of headings and paragraphs improves readability and accessibility.

    3. Text Formatting: Bold, Italic, Underline, and More


    Bold and Italic Text (< strong> and < em>): You can use the < strong> tag to make text bold for emphasis and the < em> tag to italicize text for additional emphasis. These tags also carry semantic meaning, with < strong> indicating strong importance and < em> indicating emphasis.

    Underlining Text (< u>): The < u> tag is used to underline text. However, underlining is less common in modern web design because it can be confused with hyperlinks.

    Strikethrough Text (< s>): The < s> tag adds a strikethrough line to text. It's often used to indicate that text has been deleted or is no longer relevant.

    Inline Styling (): The < span> tag is a generic container that allows you to apply inline styles to specific portions of text. This is often used in conjunction with CSS for more precise text styling.

    Typography: This topic covers more advanced text formatting options, such as controlling text alignment (left, center, right), line height (line spacing), and letter spacing (spacing between characters). These aspects contribute to the overall typography of your web page.

    4. Lists (Ordered and Unordered):


    This lesson introduces you to the concept of lists in HTML, which are crucial for organizing and presenting information effectively.

    Ordered Lists (< ol>): An ordered list is a numbered list. You use the < ol> element to create one and add individual list items with the < li> tag. Ordered lists are often used for instructions or lists with a specific sequence.

    Unordered Lists (< ul>): An unordered list is a bullet-point list. You use the < ul> element to create one and also add list items with the < li> tag. Unordered lists are suitable for items that don't require a specific order.

    List Customization: You can use CSS to customize the appearance of lists. This includes changing bullet styles, adding background colors, and controlling the spacing between list items. Customized lists can enhance the visual appeal of your web pages.
